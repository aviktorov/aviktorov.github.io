Methods

Next(String)

Advances music player to next track.

Declaration
public static void Next(string player)
Parameters
Type Name Description
System.String player

Name of the music player.
Remarks

In case of music player name collision inside a single music bank, the first music player with matched name will be played. If name collision occurs between music banks, primary music bank will be searched first.

This method will do nothing if music player doesn't have a playlist assigned to it. Use SetPlaylist(String, String) or SetPlaylist(String, String, Single) method first.

SetPlaylist(String, String)

Sets a playlist to a music player.

Declaration
public static void SetPlaylist(string player, string playlist)
Parameters
Type Name Description
System.String player

Name of the music player.
System.String playlist

Name of the playlist.
Remarks

Playlist and music player must be in the same music bank.

In case of music player name collision inside a single music bank, the first music player with matched name will be played. If name collision occurs between music banks, primary music bank will be searched first.

If music player was playing another track it'll automatically crossfade to the first track of the new playlist.
