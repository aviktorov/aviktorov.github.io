AutoAdvance

The flag that indicates if music player should auto advance to next playlist track after it finishes playing current.

Declaration
public bool AutoAdvance { get; set; }
Property Value
Type Description
System.Boolean

True, if music player is auto advancing playlist tracks.
Remarks

This flag works in pair with loop flag. There're four possible combinations:

  1. *Both flags are false* — play current track once and then stop.
  2. *Both flags are true* — play all playlist tracks in a loop and never stop.
  3. *Loop is false, auto advance is true* — play all playlist tracks once and then stop.
  4. *Loop is true, auto advance is false* — play current track in a loop and never stop.
